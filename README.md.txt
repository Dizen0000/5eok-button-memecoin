# 🕹️ 5억년 버튼 밈코인 웹사이트

이 사이트는 밈코인 "5억년 버튼(5YB)"의 랜딩 페이지입니다. 버튼을 누르면 누를수록... 뭔가가 생깁니다(?)

## 🛠️ 기술 스택
- HTML
- TailwindCSS (CDN)
- JavaScript (버튼 인터랙션)

## 🚀 배포
GitHub Pages에서 바로 배포할 수 있습니다.

## 📂 사용법
1. 이 저장소를 클론합니다.
2. `index.html`, `script.js`, `assets/meme.png` 구성 확인
3. GitHub Pages에서 `/index.html` 경로로 배포
